Cara Pakai

  1. Buka tab Akun → + Akun Baru. Isi nama, broker, currency, dan saldo awal. Saldo awal ini dipakai untuk menghitung Risk $.
  2. Buka tab Jurnal → + Trade Baru. Isi Tanggal, Market, Entry, SL, TP, Risk %, dan Hasil.
  3. Kolom Risk $, R:R, Net P/L dihitung otomatis — kamu tidak perlu isi manual.
  4. Buka tab Dashboard untuk lihat win rate, profit factor, drawdown, dan equity curve kamu.

Rumus yang Dipakai

  • Risk ($) = Risk % × saldo awal akun. Contoh: saldo $1.000, risk 1% → Risk $10.
  • R:R = jarak Entry→TP ÷ jarak Entry→SL. Contoh: TP 3× lebih jauh dari SL → R:R = 3.
  • Net P/L: Win = +(Risk × R:R) · Loss = −Risk · BE = 0.
Angka P/L di sini adalah hasil teoretis berdasarkan jarak SL:TP, bukan fill price broker. Untuk rekonsiliasi saldo riil, tetap pakai statement broker kamu.

Data & Privasi

Jurnal ini tidak punya server. Semua data trade kamu disimpan di browser milik kamu sendiri dan tidak pernah dikirim ke FNC Academy atau pihak mana pun. Admin FNC tidak bisa melihat isi jurnal kamu.

Konsekuensinya:

  • Data tidak sinkron antar perangkat. Jurnal di HP dan laptop terpisah.
  • Menghapus cache/data browser akan menghapus jurnal.
  • Mode incognito tidak menyimpan apa pun setelah tab ditutup.

Karena itu: klik Backup di tab Jurnal secara rutin (misalnya tiap akhir minggu). File backup bisa kamu Restore kapan saja, termasuk untuk pindah ke perangkat lain.
